NAME
====

**RakuFinance** - Provides routines to calculate the basis of a portfolio of investments

SYNOPSIS
========

```raku
use RakuFinance;
```

DESCRIPTION
===========

RakuFinance is a module one can use to to calculate the basis of a portfolio of investments.
